# DESIGN.md

## Identity

Mnemosyne is a cryptographic memory archive — an append-only Merkle tree service for verifiable attestations, events, documents, and workflows.

## Design Vocabulary

| Concept    | Naming        |
| ---------- | ------------- |
| leaf       | memory        |
| root       | constellation |
| checkpoint | beacon        |
| proof      | route         |
| append     | remember      |
| verify     | witness       |

## Visual Language

- maritime observatory
- cryptographic library
- ancient archives
- constellations and branching routes
- nautical charts
- immutable registry

## Anti-Patterns

- no crypto casino aesthetics
- no blockchain buzzword UI
- no neon "web3" style
- no speculative finance language

## Typography

Prefer monospace for cryptographic data (hashes, signatures, proofs).
Prefer serif for narrative content (descriptions, context).
Sans-serif for UI controls and labels.

## Colors

Deep navy, parchment, brass, starlight silver, dark slate.
Avoid neon colors, gradients, and glowing effects.
